# _handoff (drop zone)

Use this folder to drop files you want me (Atlas) to work with. Keep it organized by subfolders.

## Folders
- `ui-ux/` — Figma links, wireframes, flow diagrams, design tokens, copy docs
- `screenshots/` — UI screenshots, bugs, references
- `assets/` — logos, icons, brand assets, images, fonts
- `notes/` — requirements, TODOs, meeting notes
- `api-samples/` — Postman collections, curl examples, webhook payloads
- `db/` — schema dumps, migration notes, SQL snippets

## Conventions
- Name files with dates when possible: `2026-02-22-login-screen.png`
- If you add a Figma link, also add a short note file describing what you want changed.

## What to do next
1) Drop files into the right folder
2) Tell me which file(s) to review and what outcome you want
