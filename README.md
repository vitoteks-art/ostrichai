# OstrichAi Studio

**OstrichAi** is your all-in-one AI-powered creative platform. Generate professional videos, design stunning logos, create engaging social media content, and more with our suite of advanced AI tools.

## Key Features

- **AI Video Generation**: Create professional videos from simple text inputs.
- **UGC Video Creator**: Generate authentic user-generated content style videos.
- **Image Ad Creative**: Design high-converting ad visuals in seconds.
- **Background Remover**: Instantly remove backgrounds with precision.
- **Flyer Designer**: Create beautiful promotional materials effortlessly.
- **Social Dashboard**: Manage and generate content for all your social platforms.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```sh
   cd leads
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

### Environment Setup

Create a `.env` file in the root directory and add your API keys:
```env
VITE_KIE_API_KEY=your_kie_api_key
VITE_IMGBB_API_KEY=your_imgbb_api_key
```

To deploy the project, you can use platforms like Netlify, Vercel, or any static site hosting service.

1. Build the project:
   ```sh
   npm run build
   ```

2. Upload the `dist` folder to your hosting provider.
